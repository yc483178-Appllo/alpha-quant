## 任务说明

### 1. 数据服务健康检查 (08:45)
**目的**: 确保交易日前所有数据服务正常运行
**检查项**:
- Tushare API 连接状态
- AkShare 数据获取
- 信号服务器状态
- 飞书 Webhook 连通性

### 2. 盘前市场调研 (09:00)
**目的**: 生成盘前市场概览和热点分析
**输出**:
- 隔夜美股/港股走势
- A50 期指涨跌
- 热点板块预览
- 大盘情绪预判

### 3. 今日智能选股清单 (09:15)
**目的**: 基于多因子模型筛选今日关注标的
**输出**:
- 3-5只精选标的
- 每只标注评分、风险等级、目标价位
- 选股逻辑说明

### 4. 开盘前风险评估 (09:25)
**目的**: 开盘前最后一道风险检查
**检查项**:
- 重大政策/消息
- 外围市场异常
- 个股黑天鹅事件

### 5. 盘中持续监控 (09:30-15:00)
**目的**: 实时监控市场和持仓风险
**监控项**:
- 大盘熔断检查（跌幅>3%）
- 持仓个股止损/止盈触发
- 板块资金流向异常
- 市场情绪突变

### 6. 上午盘情绪快报 (11:30)
**目的**: 午间休市前快速总结上午走势
**输出**:
- 上午大盘回顾
- 涨跌停家数
- 热点板块排名
- 下午展望

### 7. 尾盘策略评估 (14:30)
**目的**: 收盘前评估持仓和调仓机会
**输出**:
- 持仓表现回顾
- 尾盘异动监控
- 调仓建议
- 明日预判

### 8. 收盘复盘报告 (15:35)
**目的**: 生成完整收盘复盘报告
**内容** (八个部分):
1. 今日大盘总结
2. 热点板块复盘
3. 持仓账户表现
4. 策略执行评估
5. 明日关注标的
6. 策略迭代建议
7. 明日操作计划
8. 风险提示

### 9. 数据归档备份 (16:00)
**目的**: 数据维护和归档
**任务**:
- 备份交易数据
- 归档报告文件
- 清理过期日志
- 检查磁盘空间

### 10. 龙虎榜分析 (20:00)
**目的**: 分析主力资金动向
**输出**:
- 主力资金净流入前10
- 机构席位买卖动向
- 游资热点追踪
- 明日关注提示

### 11. 周度策略复盘 (每周五 16:00)
**目的**: 总结本周策略表现
**输出**:
- 本周大盘回顾
- 策略胜率统计
- 盈亏分析
- 下周策略调整建议

### 12. 月度绩效报告 (每月末 16:00)
**目的**: 生成本月完整绩效报告
**输出**:
- 收益曲线
- 策略对比
- 风险指标（夏普比率、胜率等）
- PDF 报告
