## 快速开始

### 1. 启动看板
```bash
cd /root/.openclaw/workspace/alpha-quant
# Flask版
./quant_env/bin/python dashboard_v2.py
# 或打开独立HTML版
open static/dashboard_v2_full.html
```

### 2. 运行演示
```bash
./quant_env/bin/python chief_integration_demo.py --mode all
./quant_env/bin/python sentiment_integration_demo.py --mode all
./quant_env/bin/python optimizer_integration_demo.py --mode all
```

### 3. 使用模块
```python
from modules import (
    create_chief_agent,
    create_sentiment_pipeline,
    create_portfolio_optimizer,
    create_broker_manager,
    create_oms,
    create_continuous_learning_system
)
```
